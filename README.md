# JS BDD Frameworks Cucumber + Protractor

## Prerequisites
1. NODE version >=8.9.1
2. Git

## Install
```
npm install
```

## Run tests with npm
```
npm run e2e
npm run e2e -- --tag "@store"
```
## Run tests with gulp
```
gulp runTests
gulp explore
gulp shop
gulp sections
```
## Update webdriver
```
npm run e2e:update
```
## Start webdriver
```
npm run e2e:start-server
```
## Sort page objects by alphabetic order
```
npm run sortPO
```
